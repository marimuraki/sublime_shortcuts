## Sublime Text Shortcuts (Mac OS X): Subset of Useful Commands

|  symbol   | what?    	|
| --------- | ---------:| 
|  ⌘    	| command 	|
|  ⇧  		| shift		|
|  ^  		| control	|
|  ⌥  		| option 	|

### Go To

| how?  | what?						| 
| ----- | -------------------------:| 
| ⌘P 	| go to file 				|
| ⌘R   	| go to method				|
| ^G   	| go to lines 				|
| ^M   	| jump to matching brackets	|
| ⌥⌘←→ 	| switch to file 			|

### Select

| how? 		| what?							| 
| --------- | -----------------------------:| 
| ⌘L  		| select line 					|
| ⌘D  		| select word 					|
| ⇧⌘Space 	| select scope 					|
| ^⇧M  		| select within bracket 		|
| ⇧⌘J  		| select through indentation 	|
| ^⇧↑↓ 		| select columns 				|

### Insert

| how?  | what?					| 
| ----- | ---------------------:| 
| ⌘⇧↩︎ 	| insert line before	|
| ⌘↩︎  	| insert line after		|

### Indent

| how?  | what?	 | 
| ----- | ------:| 
| ⌘][ 	| indent |

### Delete

| how?  | what?								| 
| ----- | ---------------------------------:| 
| ^⇧K 	| delete line 						|
| ^K 	| delete to end 					|
| ⌘⌘K 	| delete from cursor to line end 	|
| ⌘⌫ 	| delete from curse to line start 	|

### Manipulate

| how?  | what?				| 
| ----- | -----------------:| 
| ⇧⌘D 	| duplicate line 	|
| ⌘J 	| join line 		|
| ^⌘↑↓ 	| swap line 		|

### Format

| how?  | what?						| 
| ----- | -------------------------:| 
| ⌘KU 	| convert to upper case 	|
| ⌘KL 	| convert to lower case		|
